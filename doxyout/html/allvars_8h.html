Detailed Description

declares global variables.

This file declares all global variables. Further variables should be added here, and declared as 'extern'. The actual existence of these variables is provided by the file 'allvars.c'. To produce 'allvars.c' from 'allvars.h', do the following:

  • Erase all #define's, typedef's, and enum's
  • add #include "allvars.h", delete the #ifndef ALLVARS_H conditional
  • delete all keywords 'extern'
  • delete all struct definitions enclosed in {...}, e.g. "extern struct global_data_all_processes {....} All;" becomes "struct global_data_all_processes All;"
